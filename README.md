#WAEC-Tracker — Pro

WAEC-Tracker — Pro is a simple, easy-to-use web app that helps students record, follow, and review their WAEC results — even without internet access.
Why it exists
Not every student has steady internet, but everyone should be able to keep an eye on their academic progress. WAEC-Tracker — Pro was built as an offline tool so students can save and check their WAEC scores anytime, anywhere.
What it does
Save your WAEC subject scores locally
Watch how your grades change over time
See your overall progress at a glance
Fully functional offline — no internet needed
Clean, responsive interface that works on phones and laptops
Built with plain HTML, CSS, and JavaScript
Why it matters
This project shows how simple web technologies can make education more accessible. By removing the need for constant internet, WAEC-Tracker — Pro helps more students keep track of their performance and stay motivated toward their goals.
